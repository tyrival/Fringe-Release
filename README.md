## Overview

Fringe is a native macOS memo application that lives in your menu bar. It combines Markdown WYSIWYG editing, color tagging, fuzzy search, and multi-tier local backup — all in a minimal, distraction-free interface.

**Free. No ads. Native Apple experience. Requires macOS 14+.**

## Features

### Markdown WYSIWYG Editor
Write with bold, italic, strikethrough, and inline code — all rendered in real-time. Toggle formatting with keyboard shortcuts (⌘B, ⌘I, ⌘K, ⌘⇧X) or the floating toolbar. Bold and italic combine seamlessly into ***bold-italic***.

### Color Tagging
Each card supports a left accent color bar and tinted background. Choose from 7 palette colors or follow system appearance. Perfect for visually distinguishing cards at a glance.

### Reliable Backup
Automatically back up your data with hourly snapshots for the past 24 hours, daily backups for 7 days, and weekly archives for 6 months. Plain JSON format, stored locally. Browse and restore any version with one click.

### Fuzzy Search
Type in the search bar to filter cards in real-time. Cards matching your query appear instantly, empty search shows all. Zero configuration — just type and find.
