## 概要

Fringe は、macOS のメニューバーに常駐するネイティブメモアプリです。Markdown WYSIWYG 編集、カラータグ、あいまい検索、多層ローカルバックアップをミニマルで集中できるインターフェースに統合しています。

**無料・広告なし・Apple ネイティブ体験。macOS 14+ 対応。**

## 主な機能

### Markdown WYSIWYG エディタ
太字、斜体、打ち消し線、インラインコード — すべてリアルタイムでレンダリング。キーボードショートカット（⌘B、⌘I、⌘K、⌘⇧X）またはフローティングツールバーで書式を切り替え。太字と斜体はシームレスに組み合わせて ***太字斜体*** に。

### カラータグ
各カードに左側のアクセントカラーバーと色付き背景を設定可能。7色のパレットから選択、またはシステムの外観モードに追従。一目でカードを視覚的に区別できます。

### 信頼のバックアップ
過去24時間の毎時スナップショット、7日間の日次バックアップ、6ヶ月間の週次アーカイブでデータを自動バックアップ。プレーンJSON形式でローカルに保存。任意のバージョンを閲覧してワンクリックで復元。

### あいまい検索
検索バーに入力するとカードがリアルタイムにフィルタリング。クエリに一致するカードが即座に表示され、検索欄が空のときはすべて表示。設定不要 — 入力するだけですぐに検索。
