## 概述

Fringe 是一款常驻 macOS 菜单栏的原生便签应用。融合 Markdown 所见即所得编辑、颜色标签、模糊搜索与多级本地备份于极简无干扰的界面中。

**免费、无广告、Apple 原生体验。要求 macOS 14+。**

## 核心功能

### Markdown 所见即所得编辑器
支持粗体、斜体、删除线、行内代码，实时渲染。快捷键 ⌘B / ⌘I / ⌘K / ⌘⇧X 一键切换样式，悬浮工具栏点击亦可。粗体和斜体可叠加为 ***粗斜体***。

### 颜色标签
每张卡片支持左侧彩色边框和浅色背景。7 种预设颜色可选，或跟随系统外观。一眼区分不同类型的便签内容。

### 可靠备份
自动备份数据：过去 24 小时每小时快照，过去 7 天每天一份，过去 6 个月每周一份。明文 JSON 格式，本地存储。浏览历史版本，一键恢复。

### 模糊搜索
在搜索栏输入关键词，卡片实时过滤。清空搜索框即刻恢复全部卡片。无需配置，即输即搜。
