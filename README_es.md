## Descripción general

Fringe es una aplicación de notas nativa para macOS que vive en tu barra de menús. Combina edición Markdown WYSIWYG, etiquetas de color, búsqueda difusa y respaldo local multinivel en una interfaz mínima y sin distracciones.

**Gratis. Sin anuncios. Experiencia nativa Apple. Requiere macOS 14+.**

## Funciones

### Editor Markdown WYSIWYG
Escribe con negrita, cursiva, tachado y código en línea — todo renderizado en tiempo real. Alterna el formato con atajos de teclado (⌘B, ⌘I, ⌘K, ⌘⇧X) o la barra de herramientas flotante. La negrita y la cursiva se combinan perfectamente en ***negrita-cursiva***.

### Etiquetas de color
Cada tarjeta admite una barra de color de acento a la izquierda y un fondo tintado. Elige entre 7 colores de paleta o sigue la apariencia del sistema. Perfecto para distinguir visualmente las tarjetas de un vistazo.

### Respaldo confiable
Respaldos automáticos con instantáneas cada hora de las últimas 24 horas, copias diarias de 7 días y archivos semanales de 6 meses. Formato JSON simple, almacenado localmente. Explora y restaura cualquier versión con un solo clic.

### Búsqueda difusa
Escribe en la barra de búsqueda para filtrar tarjetas en tiempo real. Las tarjetas que coinciden con tu consulta aparecen al instante. Sin búsqueda se muestran todas. Cero configuración — solo escribe y encuentra.
